# Тестирование функционала проверки треугольника

## 📋 О проекте
Данный репозиторий содержит комплект тестовой документации для проверки веб-формы, определяющей тип треугольника на основе **правила неравенства треугольника** (сумма длин любых двух сторон должна быть больше длины третьей стороны).

**Цель тестирования:** Проверка корректности валидации вводимых данных, вычисления результата и обработки граничных значений.

## 📂 Структура репозитория
- **`Тест-план Форма проверки треугольников.docx`** — Детальный тест-план (объект тестирования, критерии входа/выхода, риски, стратегия тестирования).
- **`Тест-кейсы Форма проверки треугольников.xlsx`** — Единый файл с тест-кейсами (позитивные, негативные, граничные значения).
- **`Баг-репорт Формы проверки треугольника.xlsx`** — Отчет о найденных дефектах (с приоритетами и шагами воспроизведения).

> *Примечание:* Все файлы находятся в корне репозитория для удобства навигации.
